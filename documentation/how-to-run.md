## React application

Run build environment:
`npm run build`
Run development environment:
`npm run dev`
Run tests:
`npm run test`
Run test coverage:
`npm run coverage`


## Express service
Run production environment:
`npm run start-prod`
For the following, append -win if running on windows machines:
Run development environment:
`npm run start`
Run tests:
`npm run test`
Run test coverage:
`npm run coverage`
